# TCP chat
### Go Programming Language - chapter 8.10

## run project

> 1: compile code
```console
go build cmd/start/main.go
```

> 2: run
```console
go run ./main
```

> 3: connect tcp clients to localhost 8000
>> sugestion: use netcat as tcp client
>>> open at least 2 terminals and hit:
```console
nc localhost 8000
```

> 4: keep typing and have fun
